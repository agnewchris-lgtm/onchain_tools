# KOSPI6900 viral-source meme pattern (2026-05-04)

Use this as a compact case reference for tokens spawned from a viral non-token finance meme.

## Alert shape

- Token: `$KOSPI6900` / `코스피6900` on Solana
- Source social: `https://x.com/jukan05/status/2051223306628735158` from @jukan05 (~99.7k followers)
- Source tweet metrics at research time: ~2.4k likes, ~216k views, 334 RTs
- Token metrics moved fast: alert MC ~$34.5k, later ~$19.7k; volume grew from ~$135k to ~$170k in minutes; liquidity ~$10-14k

## What the source actually was

The source tweet was a quote/image meme about crypto traders rotating into Korean stocks / KOSPI. The tweet text was only a media URL; no CA, no ticker, no explicit token endorsement. The quoted tweet was Robert Leshner joking: “My crypto friends are all trading Korean stocks at 11pm on a Sunday.”

Vision on the media showed a Wojak leaving chaotic “CRYPTO” into a bright “KOSPI” doorway with KRX/Korean blue-chip references. This supports the narrative, but not dev/team confirmation.

## X findings

- CA-specific search showed mostly bot/call-account posts with tiny engagement.
- Search also surfaced multiple competing `KOSPI6900` / `KRX` CAs within minutes.
- Generic `KOSPI` search contained legitimate market/news chatter, which can pollute token relevance scoring.

## Verdict pattern

Classify as **WATCH**, not APE/BUY, when:

- Pure meme/no product/team, but source meme is genuinely viral and early token volume is exceptional for tiny MC.
- Source account did not post the CA/ticker, so treat as viral narrative only.
- Competing CAs and botty social proof are present.

Reserve **APE/BUY** for cases where the high-follower source explicitly endorses/posts the CA or there is real product/team confirmation.
